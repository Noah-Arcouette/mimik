# Name

logname, whoami - Get current user's login name


# Synopsis

`logname`

`whoami`


# Description

*whoami* shall be effectively the same as *logname*.

*logname* shall print the current user's login name.


# Options

Options shall no be considered.


# Operands

Operands shall no be considered.


# Environment Variables

Environment variable shall not effect the operation of *logname*.


# StdIn

Standard input shall not be used.


# StdOut

Standard output shall be used to print the current user's login name.


# StdErr

Standard error shall not be used.


# Input Files

No input files shall be used.


# Output Files

*logname* produces no outputs.


# Exit Status

*logname* shall never fail.


# Extended Description

There is no extended description.

There are no future directions.
