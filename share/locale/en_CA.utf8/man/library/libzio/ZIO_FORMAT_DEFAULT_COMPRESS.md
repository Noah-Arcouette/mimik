# Name

ZIO_FORMAT_DEFAULT_COMPRESS -- The default compression algorithm


# Synopsys

```c
#include <zio.h>

#define ZIO_FORMAT_DEFAULT_COMPRESS 0
```


# Description

This format is set at compilation time for Mimik. And shall always be a
compressed format. It is the default compression method by the system.
