# Synopsis

`.o` - binary object file. A file usually produced by the `as` or `cc` utility,
that is not yet executable or part of a defined use. Common formats:
 - MiO - Mimik
 - ELF - Linux, Unix*, and BSD* systems
 - PE32+ - Windows
 - Macho - MacOS and other Darwin based systems
